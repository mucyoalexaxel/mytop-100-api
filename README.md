# MyTop100Movies API
## Description

MyTop100Movies - API, is an application which lets users set their top 100 movies. Movies could come from an API like The Movie Database. It allows a user to add a movie, list and rank their movie selections.
## InstallationS

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
